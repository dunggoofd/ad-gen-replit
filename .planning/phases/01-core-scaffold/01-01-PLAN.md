<task type="auto">
  <name>Task 2: Create server.js with full middleware stack and HTML fallback route</name>
  <files>server.js, public/.gitkeep</files>
  <action>
Create `server.js` as the Express 5 entry point. Create `public/` directory with a `.gitkeep` placeholder (Multer upload dirs will be added here in Phase 3).

`server.js` must:

1. Require: `express`, `cookie-parser`, `cors`, `path`, and `./database/init` (the pool+initDb export created in plan 01-02).
2. Wire middleware in this order:
   - `app.use(cors())`
   - `app.use(express.json())`
   - `app.use(express.urlencoded({ extended: false }))` â€” explicit, Express 5 changed the default
   - `app.use(cookieParser())`
   - `app.use(express.static(path.join(__dirname, 'public')))`
3. Mount routes (health route added in this same file for now; will be extracted to routes/health.js in plan 01-02):
   - `app.get('/api/health', ...)` â€” placeholder that returns `{ status: 'ok', uptime: process.uptime() }`
   - `app.get('/*splat', ...)` â€” serves `views/index.html` (Express 5 named wildcard â€” NOT `/*`)
4. Add a 4-argument error handler: `app.use((err, req, res, next) => { ... })`
5. Define `async function start()` that:
   - Calls `await initDb()` before `app.listen()`
   - Calls `app.listen(PORT, () => console.log(...))`
   - PORT = `process.env.PORT || 3000`
6. Call `start().catch(err => { console.error('[fatal] Startup failed:', err); process.exit(1); })`

Key Express 5 rules to follow (from research):
- Wildcard route MUST be `'/*splat'` not `'/*'` â€” Express 5 path-to-regexp v8 requires named wildcards
- Use `res.status(N).json(body)` not `res.send(body, status)`
- Do NOT use `app.del()` â€” use `app.delete()`

Load dotenv at the very top with: `if (process.env.NODE_ENV !== 'production') require('dotenv').config();`

Add `pool.on('error', ...)` listener â€” but this is done inside database/init.js (plan 01-02); server.js just calls `initDb()`.
  </action>

    <automated>node -e "const app = require('./server'); console.log('require OK')" 2>&1 | grep -v "listen\|EADDRINUSE\|initDb\|DATABASE" || true; node --check server.js && echo "syntax OK"</automated>
  </verify>
  <done>
`server.js` passes `node --check` (syntax valid), requires cleanly, has all 5 middleware lines, the `/*splat` catch-all route, and the 4-arg error handler. `public/` directory exists.
  </done>
</task>
